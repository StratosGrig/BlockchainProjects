Task 1

For the first task you may input the following arguments :

1) pubkey 
2) -param ( a future time expressed either in block height or in UNIX Epoch time)

For example you can run the following in cmd : python Bitcoin_Task1.py cRvyLwCPLU88jsyj94L7iJjQX5C2f8koG4G2gevN4BeSGcEvfKe9 -param 2000

Task 2

For the second task you should input the following arguments :

1) private key
2) -param 
3) -to_address

For example you can run the following in cmd : python Bitcoin_Task2.py 
cRvyLwCPLU88jsyj94L7iJjQX5C2f8koG4G2gevN4BeSGcEvfKe9 -param 2000 -to_address n4bkvTyU1dVdzsrhWBqBw8fEMbHjJvtmJR
